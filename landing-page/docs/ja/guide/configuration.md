# 設定

Copilot Island は最小限の設定で、すぐに使用できます。

## 設定パネル

展開されたノッチパネルで**⋯ メニューアイコン**をクリックし、**設定**を選択してアクセスします。

### サウンドエフェクト

完了チャイムと失敗サウンドのオン/オフを切り替えます：

- **完了サウンド** — エージェントがタスクを完了したとき（`session.task_complete` → `assistant.turn_end`）に再生
- **失敗サウンド** — `abort`、`session.error`、または予期しない `session.shutdown` 時に再生

### About とアップデート

設定の **About** タブには現在のアプリバージョンが表示され、**アップデートを確認**（Sparkle を利用）できます。Copilot Island は GitHub に新しいリリースが公開されると通知します。

## セッション監視

Copilot Island は自動的に `~/.copilot/session-state/` の変更を監視します。  
設定は不要 — そのまま動作します。

起動時には、UI を高速に保つため**現在のタスク**（最後の `user.message` 以降のイベント）のみが読み込まれます。古い履歴は再生されません。

## macOS の権限

初回起動時、プロンプトが表示されたらこの権限を付与してください：

- **ファイルとフォルダ** → ホームディレクトリ内の Copilot CLI セッションファイルの読み取りアクセスを許可
