# クイックスタート

## 1. Copilot セッションを開始する

ターミナルを開き、任意の Copilot CLI コマンドを実行します：

```bash
copilot "この関数を async/await を使うようにリファクタリングして"
```

## 2. ノッチを確認する

Copilot Island が新しいセッションを**自動的に**検出します。以下が表示されます：

- アクティブなセッションを示すノッチ内の**点滅するドット**
- エージェントの現在の状態（作業中・完了・エラー）を示す左ピークアイコンのアニメーション

## 3. パネルを確認する

**ノッチをクリック**して Copilot Island パネルを展開します：

| セクション | 表示内容 |
|-----------|----------|
| 🔵 セッション | すべてのアクティブおよび最近の Copilot CLI セッション |
| 💬 チャット | Markdown レンダリング付きの完全な会話履歴 |
| ⚡ ツール | リアルタイムのツール実行フィード（ファイル書き込み、シェルコマンド） |
| ⚙️ 設定 | サウンド、About、アップデートの確認 |

## 4. セッション通知

エージェントがタスクを完了すると：
- 🏆 左ノッチピークに**トロフィーアニメーション**が 3 秒間表示される
- 🎵 **完了チャイム**が鳴る（設定で無効化可能）

エージェントが中断されたりエラーが発生した場合：
- ❌ **失敗アニメーション**が 3 秒間点滅する
- 🔔 **失敗サウンド**が鳴る

## 5. キーボードショートカット

| ショートカット | 操作 |
|--------------|------|
| ノッチをクリック | 展開/折りたたみの切り替え |
| ノッチにホバー | アクティブなセッションをプレビュー |
