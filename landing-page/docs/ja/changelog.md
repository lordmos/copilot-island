# 変更履歴

Copilot Island のすべての主な変更点をここに記録します。

---

## v0.1.8 — 2026-04-07

### 新機能
- **Sparkle による自動アップデート** — Copilot Island が Sparkle フレームワークを使用して自動的にアップデートを確認するようになりました。アップデートは GitHub Pages 経由で安全に配信されます。

### 修正
- アプリバージョン表示とメニューレイアウトの幅調整
- 信頼性の高いアップデート検出のための Sparkle `SUFeedURL` 設定

---

## v0.1.7 — 2026-04-06

このリリースはノッチ UI、イベントシステム、サウンドエフェクトの大規模な刷新です。

### 新機能
- **ピークステートマシン** — 左ピークアイコンがセッション状態に応じてアニメーション：アイドル時の呼吸、作業中のパルス、タスク完了時に 3 秒間トロフィー 🏆、中断/エラー/シャットダウン時に 3 秒間失敗 ❌
- **8 ビットサウンドエフェクト** — タスク完了時に上昇チャイム、失敗時に下降矩形波。サウンドは起動時から 5 秒以内のライブイベントにのみ再生
- **失敗サウンド** — `abort`、`session.error`、`session.shutdown` イベントに対する個別のサウンド
- **アシスタントメッセージの結合** — 同じターン内のアシスタント出力が複数の断片ではなく、1 つの読みやすいメッセージブロックにまとめられる
- **起動フィルター** — 起動時に現在のタスク（最後の `user.message` 以降）のみを読み込み、セッション履歴の完全な再生を回避
- **アップデートを確認** — 設定パネルに Sparkle を利用したバージョン確認ボタン付きの「About」ページを追加
- **設定の再設計** — よりクリーンなレイアウト、修正されたトグル色、設定全体でのテーマカラーの適切な使用

### 修正
- **パネルが画面上部に揃えられる** — 展開されたノッチパネルが物理的な画面の端にフラッシュして表示される（ノッチの下にギャップなし）
- **起動時のちらつきを排除** — 起動時にセッションリストが数百回再レンダリングされなくなった。バッチ処理が単一の状態更新を発行
- **サウンドタイミングを修正** — 完了サウンドは `session.task_complete` 自体ではなく、その後の `assistant.turn_end` で発火するように
- **リアルタイムチャット更新** — セッション詳細ビューが毎秒新しいメッセージをポーリング
- **動的ノッチ寸法** — パネルの幅と高さを実際の macOS スクリーン API（`auxiliaryTopLeftArea` / `auxiliaryTopRightArea`）から計算
- **統一されたノッチバー** — 閉じた状態では左アイコン、中央のドット、右のセッション数を覆う 1 本のシームレスなブラックバーを表示
- **カラー監査** — 読みにくいプレースホルダーテキスト色、一貫性のないトグルアクセント色、設定内のブランド外の青色トーンを修正

## v0.1.0 — 2026-03-31

🎉 Copilot Island の**初回リリース**。

### 新機能
- `~/.copilot/session-state/` をリアルタイムで監視するノッチオーバーレイウィンドウ
- すべてのアクティブおよび最近の Copilot CLI セッションを表示するセッションリスト
- Markdown レンダリング、コードブロック、ツール結果付きのチャット履歴ビュー
- 左ステータスアイコンと右セッション数を持つ折りたたまれた「ピル」状態
- ノッチのクリックまたはホバーで展開/折りたたみ
- Copilot インスパイアのダークセージグリーンテーマ
- 画面の端にフラッシュしたフラットトップのノッチシェイプ、丸みを帯びた下角
- エージェントのタスク完了時のサウンドエフェクト
- macOS 14.0 以降（Sonoma）、MacBook ノッチのサポート
- Apache 2.0 オープンソースライセンス
