# 빠른 시작

## 1. Copilot 세션 시작하기

터미널을 열고 Copilot CLI 명령을 실행합니다:

```bash
copilot "이 함수를 async/await를 사용하도록 리팩터링해줘"
```

## 2. 노치 확인하기

Copilot Island가 새 세션을 **자동으로** 감지합니다. 다음이 표시됩니다:

- 활성 세션을 나타내는 노치 내 **깜박이는 점**
- 에이전트의 현재 상태（작업 중, 완료, 오류）를 보여주는 왼쪽 피크 아이콘 애니메이션

## 3. 패널 탐색하기

**노치를 클릭**하여 Copilot Island 패널을 펼칩니다:

| 섹션 | 표시 내용 |
|------|-----------|
| 🔵 세션 | 모든 활성 및 최근 Copilot CLI 세션 |
| 💬 채팅 | Markdown 렌더링이 적용된 전체 대화 기록 |
| ⚡ 도구 | 실시간 도구 실행 피드（파일 쓰기, 셸 명령） |
| ⚙️ 설정 | 사운드, About, 업데이트 확인 |

## 4. 세션 알림

에이전트가 작업을 완료하면:
- 🏆 왼쪽 노치 피크에 **트로피 애니메이션**이 3초간 표시됨
- 🎵 **완료 차임**이 재생됨（설정에서 비활성화 가능）

에이전트가 중단되거나 오류가 발생한 경우:
- ❌ **실패 애니메이션**이 3초간 깜박임
- 🔔 **실패 사운드**가 재생됨

## 5. 키보드 단축키

| 단축키 | 동작 |
|--------|------|
| 노치 클릭 | 펼치기/접기 전환 |
| 노치 호버 | 활성 세션 미리보기 |
