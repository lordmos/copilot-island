# 변경 사항

Copilot Island의 모든 주요 변경 사항이 여기에 기록됩니다.

---

## v0.1.8 — 2026-04-07

### 새 기능
- **Sparkle를 통한 자동 업데이트** — Copilot Island가 Sparkle 프레임워크를 사용하여 자동으로 업데이트를 확인합니다. 업데이트는 GitHub Pages를 통해 안전하게 제공됩니다.

### 수정
- 앱 버전 표시 및 메뉴 레이아웃 너비 조정
- 신뢰할 수 있는 업데이트 감지를 위한 Sparkle `SUFeedURL` 구성

---

## v0.1.7 — 2026-04-06

이 릴리스는 노치 UI, 이벤트 시스템, 사운드 효과의 대대적인 개편입니다.

### 새 기능
- **피크 상태 머신** — 왼쪽 피크 아이콘이 세션 상태에 따라 애니메이션됩니다: 유휴 시 호흡, 작업 중 펄스, 작업 완료 시 3초간 트로피 🏆, 중단/오류/종료 시 3초간 실패 ❌
- **8비트 사운드 효과** — 작업 완료 시 상승 차임, 실패 시 하강 구형파. 시작 시 5초 미만의 라이브 이벤트에만 재생
- **실패 사운드** — `abort`, `session.error`, `session.shutdown` 이벤트에 대한 고유한 사운드
- **어시스턴트 메시지 연결** — 같은 턴 내의 어시스턴트 출력이 여러 조각 대신 하나의 읽기 쉬운 메시지 블록으로 합쳐짐
- **시작 필터** — 실행 시 현재 작업（마지막 `user.message` 이후）만 로드하여 전체 세션 기록 재생 방지
- **업데이트 확인** — 설정 패널에 Sparkle 기반 버전 확인 버튼이 있는 "About" 페이지 추가
- **설정 재설계** — 더 깔끔한 레이아웃, 수정된 토글 색상, 설정 전체에 적절한 테마 색상 사용

### 수정
- **패널이 화면 상단에 정렬** — 펼쳐진 노치 패널이 물리적 화면 가장자리에 맞닿아 시작됨（노치 아래에 간격 없음）
- **시작 시 깜박임 제거** — 시작 시 세션 목록이 수백 번 재렌더링되지 않도록 수정. 일괄 처리가 단일 상태 업데이트를 발생
- **사운드 타이밍 수정** — 완료 사운드가 `session.task_complete` 이벤트 자체가 아닌 이후 `assistant.turn_end`에서 발생
- **실시간 채팅 업데이트** — 세션 상세 보기가 매초 새 메시지를 폴링
- **동적 노치 치수** — 실제 macOS 화면 API（`auxiliaryTopLeftArea` / `auxiliaryTopRightArea`）에서 패널 너비와 높이 계산
- **통합된 노치 바** — 닫힌 상태에서 왼쪽 아이콘, 중앙 점, 오른쪽 세션 수를 덮는 하나의 매끄러운 검은 바 표시
- **색상 감사** — 읽기 어려운 플레이스홀더 텍스트 색상, 일관성 없는 토글 액센트 색상, 설정의 브랜드 외 파란색 톤 수정

## v0.1.0 — 2026-03-31

🎉 Copilot Island **초기 릴리스**.

### 새 기능
- `~/.copilot/session-state/`를 실시간으로 모니터링하는 노치 오버레이 창
- 모든 활성 및 최근 Copilot CLI 세션을 표시하는 세션 목록
- Markdown 렌더링, 코드 블록, 도구 결과가 포함된 채팅 기록 뷰
- 왼쪽 상태 아이콘과 오른쪽 세션 수가 있는 접힌 "필" 상태
- 노치 클릭 또는 호버로 펼치기/접기
- Copilot에서 영감받은 다크 세이지 그린 테마
- 화면 가장자리에 맞닿은 플랫 탑 노치 형태, 둥근 하단 모서리
- 에이전트 작업 완료 시 사운드 효과
- macOS 14.0 이상（Sonoma）, MacBook 노치 지원
- Apache 2.0 오픈 소스 라이선스
