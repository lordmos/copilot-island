# 快速開始

## 1. 開始一個 Copilot 工作階段

開啟終端機並執行任何 Copilot CLI 指令：

```bash
copilot "將此函式重構為使用 async/await"
```

## 2. 觀察瀏海

Copilot Island 將**自動**偵測到新的工作階段。您將看到：

- 瀏海中出現一個**脈衝圓點**，表示有活動中的工作階段
- 左側預覽圖示動畫顯示代理程式的目前狀態（工作中、完成、錯誤）

## 3. 探索面板

**點擊瀏海**以展開 Copilot Island 面板：

| 區域 | 顯示內容 |
|------|---------|
| 🔵 工作階段 | 所有活動中及近期的 Copilot CLI 工作階段 |
| 💬 對話 | 完整對話歷史，支援 Markdown 渲染 |
| ⚡ 工具 | 即時工具執行記錄（檔案寫入、Shell 指令） |
| ⚙️ 設定 | 音效、關於資訊及檢查更新 |

## 4. 工作階段通知

當代理程式完成任務時：
- 🏆 左側瀏海預覽中**獎盃動畫**閃爍 3 秒
- 🎵 播放**完成提示音**（可在「設定」中關閉）

若代理程式被中止或遇到錯誤：
- ❌ **失敗動畫**閃爍 3 秒
- 🔔 播放**失敗音效**

## 5. 鍵盤快捷鍵

| 快捷鍵 | 操作 |
|--------|------|
| 點擊瀏海 | 切換展開/折疊 |
| 懸停瀏海 | 預覽活動中的工作階段 |
