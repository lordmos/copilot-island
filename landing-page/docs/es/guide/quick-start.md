# Inicio Rápido

## 1. Inicia una Sesión de Copilot

Abre tu terminal y ejecuta cualquier comando de Copilot CLI:

```bash
copilot "Refactoriza esta función para usar async/await"
```

## 2. Observa la Muesca

Copilot Island detecta **automáticamente** la nueva sesión. Verás:

- Un **punto pulsante** en la muesca que indica una sesión activa
- El ícono de vista previa izquierdo se anima para mostrar el estado actual del agente (trabajando, completado, error)

## 3. Explora el Panel

**Haz clic en la muesca** para expandir el panel de Copilot Island:

| Sección | Qué ves |
|---------|---------|
| 🔵 Sesiones | Todas las sesiones activas y recientes del Copilot CLI |
| 💬 Chat | Historial completo de conversación con renderizado Markdown |
| ⚡ Herramientas | Feed de ejecución de herramientas en tiempo real (escrituras de archivos, comandos de shell) |
| ⚙️ Ajustes | Sonido, Acerca de y buscar actualizaciones |

## 4. Notificaciones de Sesión

Cuando el agente termina una tarea:
- 🏆 Una **animación de trofeo** destella en la vista previa izquierda de la muesca durante 3 segundos
- 🎵 Un **tono de finalización** suena (se puede desactivar en Ajustes)

Si el agente es cancelado o encuentra un error:
- ❌ Una **animación de fallo** destella durante 3 segundos
- 🔔 Un **sonido de fallo** suena

## 5. Atajos de Teclado

| Atajo | Acción |
|-------|--------|
| Clic en la muesca | Expandir/colapsar |
| Pasar el cursor sobre la muesca | Vista previa de la sesión activa |
