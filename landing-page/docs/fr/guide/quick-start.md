# Démarrage rapide

## 1. Démarrer une session Copilot

Ouvrez votre terminal et exécutez n'importe quelle commande Copilot CLI :

```bash
copilot "Refactor this function to use async/await"
```

## 2. Observer l'encoche

Copilot Island **détecte automatiquement** la nouvelle session. Vous verrez :

- Un **point clignotant** dans l'encoche indiquant une session active
- L'icône d'aperçu gauche s'anime pour afficher l'état actuel de l'agent (en cours, terminé, erreur)

## 3. Explorer le panneau

**Cliquez sur l'encoche** pour développer le panneau Copilot Island :

| Section | Ce que vous voyez |
|---------|------------------|
| 🔵 Sessions | Toutes les sessions Copilot CLI actives et récentes |
| 💬 Chat | Historique complet des conversations avec rendu Markdown |
| ⚡ Outils | Flux d'exécution des outils en temps réel (écritures de fichiers, commandes shell) |
| ⚙️ Paramètres | Son, À propos et vérification des mises à jour |

## 4. Notifications de session

Lorsque l'agent termine une tâche :
- 🏆 Une **animation de trophée** s'affiche dans l'aperçu gauche de l'encoche pendant 3 secondes
- 🎵 Une **sonnerie de fin** retentit (désactivable dans les Paramètres)

Si l'agent est interrompu ou rencontre une erreur :
- ❌ Une **animation d'échec** s'affiche pendant 3 secondes
- 🔔 Un **son d'échec** retentit

## 5. Raccourcis clavier

| Raccourci | Action |
|-----------|--------|
| Cliquer sur l'encoche | Basculer développer/réduire |
| Survoler l'encoche | Aperçu de la session active |
