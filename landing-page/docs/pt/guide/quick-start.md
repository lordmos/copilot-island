# Início Rápido

## 1. Inicie uma Sessão do Copilot

Abra seu terminal e execute qualquer comando do Copilot CLI:

```bash
copilot "Refatore esta função para usar async/await"
```

## 2. Observe o Notch

O Copilot Island detecta **automaticamente** a nova sessão. Você verá:

- Um **ponto pulsante** no notch indicando uma sessão ativa
- O ícone de prévia esquerdo anima para mostrar o estado atual do agente (trabalhando, concluído, erro)

## 3. Explore o Painel

**Clique no notch** para expandir o painel do Copilot Island:

| Seção | O que você vê |
|-------|---------------|
| 🔵 Sessões | Todas as sessões ativas e recentes do Copilot CLI |
| 💬 Chat | Histórico completo de conversa com renderização Markdown |
| ⚡ Ferramentas | Feed de execução de ferramentas em tempo real (gravações de arquivos, comandos shell) |
| ⚙️ Configurações | Som, Sobre e verificação de atualizações |

## 4. Notificações de Sessão

Quando o agente conclui uma tarefa:
- 🏆 Uma **animação de troféu** pisca na prévia esquerda do notch por 3 segundos
- 🎵 Um **toque de conclusão** toca (pode ser desativado nas Configurações)

Se o agente for cancelado ou encontrar um erro:
- ❌ Uma **animação de falha** pisca por 3 segundos
- 🔔 Um **som de falha** toca

## 5. Atalhos de Teclado

| Atalho | Ação |
|--------|------|
| Clique no notch | Expandir/recolher |
| Passar o cursor sobre o notch | Prévia da sessão ativa |
