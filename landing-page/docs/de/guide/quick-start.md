# Schnellstart

## 1. Eine Copilot-Sitzung starten

Öffnen Sie Ihr Terminal und führen Sie einen beliebigen Copilot CLI-Befehl aus:

```bash
copilot "Refactor this function to use async/await"
```

## 2. Den Notch beobachten

Copilot Island **erkennt die neue Sitzung automatisch**. Sie sehen:

- Einen **pulsierenden Punkt** im Notch, der eine aktive Sitzung anzeigt
- Das linke Vorschau-Symbol animiert den aktuellen Zustand des Agenten (aktiv, abgeschlossen, Fehler)

## 3. Das Panel erkunden

**Klicken Sie auf den Notch**, um das Copilot Island-Panel zu erweitern:

| Bereich | Was Sie sehen |
|---------|--------------|
| 🔵 Sitzungen | Alle aktiven und aktuellen Copilot CLI-Sitzungen |
| 💬 Chat | Vollständiger Gesprächsverlauf mit Markdown-Darstellung |
| ⚡ Werkzeuge | Echtzeit-Feed zur Werkzeugausführung (Datei-Schreibvorgänge, Shell-Befehle) |
| ⚙️ Einstellungen | Sound, Info und Update-Prüfung |

## 4. Sitzungsbenachrichtigungen

Wenn der Agent eine Aufgabe abschließt (Aufgabe abgeschlossen):
- 🏆 Eine **Trophäen-Animation** erscheint im linken Notch-Vorschaubereich für 3 Sekunden
- 🎵 Ein **Abschluss-Chime** ertönt (kann in den Einstellungen deaktiviert werden)

Wenn der Agent abgebrochen wird oder einen Fehler hat:
- ❌ Eine **Fehler-Animation** erscheint für 3 Sekunden
- 🔔 Ein **Fehlerton** ertönt

## 5. Tastaturkürzel

| Kürzel | Aktion |
|--------|--------|
| Notch klicken | Panel ein-/ausklappen |
| Notch hovern | Vorschau der aktiven Sitzung |
