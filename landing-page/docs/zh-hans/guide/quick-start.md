# 快速开始

## 1. 开始一个 Copilot 会话

打开终端并运行任意 Copilot CLI 命令：

```bash
copilot "将此函数重构为使用 async/await"
```

## 2. 观察刘海

Copilot Island 将**自动**检测到新会话。您将看到：

- 刘海中出现一个**脉冲圆点**，表示有活动会话
- 左侧预览图标动画显示代理的当前状态（工作中、完成、错误）

## 3. 探索面板

**点击刘海**以展开 Copilot Island 面板：

| 区域 | 显示内容 |
|------|---------|
| 🔵 会话 | 所有活动和近期的 Copilot CLI 会话 |
| 💬 对话 | 完整对话历史，支持 Markdown 渲染 |
| ⚡ 工具 | 实时工具执行日志（文件写入、Shell 命令） |
| ⚙️ 设置 | 音效、关于信息及检查更新 |

## 4. 会话通知

当代理完成任务时：
- 🏆 左侧刘海预览中**奖杯动画**闪烁 3 秒
- 🎵 播放**完成提示音**（可在「设置」中关闭）

若代理被中止或遇到错误：
- ❌ **失败动画**闪烁 3 秒
- 🔔 播放**失败音效**

## 5. 键盘快捷键

| 快捷键 | 操作 |
|--------|------|
| 点击刘海 | 切换展开/折叠 |
| 悬停刘海 | 预览活动会话 |
